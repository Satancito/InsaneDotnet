# InsaneDotnet
Insane library Compatible with .NET 10 and upwards.

## Documentation

- [Docs/Cryptography.md](Docs/Cryptography.md)

## Add to your project

**Package Reference**   
```
<PackageReference Include="InsaneIO.Insane" Version="10.3.0" />
```

**Dotnet CLI**   
```
dotnet add package InsaneIO.Insane --version 10.3.0
```

**Package Manager**   
```
Install-Package InsaneIO.Insane -Version 10.3.0
```
